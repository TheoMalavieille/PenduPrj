# Jeu du Pendu - Projet MGA802

Jeu réalisé dans le cours MGA802 - Introduction à la programmation avec Python à l'ÉTS

## Description
Ce programme permet de jouer au jeu du pendu dans le terminal. L’ordinateur choisit un mot au hasard dans un fichier texte et le joueur doit le deviner avant d'épuiser ses 6 chances. 

Le programme respecte les consignes de normalisation des accents pour permettre une expérience de jeu fluide.


## Contenu du dépôt
Le dépôt contient les fichiers suivants :
- PenduGame.py : Le programme principal du jeu.
- mots_pendu.txt : La liste de mots utilisée par défaut.
- README.md : Documentation du projet.


## Fonctionnalités
- Gestion des fichiers : Utilisation d'un fichier personnalisé ou du fichier par défaut si nécessaire.
- Normalisation des accents : Le joueur peut proposer une lettre simple pour trouver une lettre accentuée.
- Système d'indice : Une lettre absente du mot est révélée lorsqu'il ne reste qu'une seule chance.
- Interface robuste : Gestion des erreurs de saisie (chiffres, lettres multiples, etc.).


## Comment utiliser le programme
1. Télécharger les fichiers du projet.
2. Ouvrir le dossier dans un terminal.
3. Exécuter le programme avec la commande : python PenduGame.py
4. Suivre les instructions affichées à l’écran.
